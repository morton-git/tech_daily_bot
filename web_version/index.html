使用指南
🚀 Step 1：獲取 Gemini API Key
  1. 訪問 https://aistudio.google.com/apikey
  2. 登入您的 Google 帳號
  3. 點擊 "Create API Key" 按鈕
  4. 複製生成的 API Key (類似：AIzaSy...)
🔑 Step 2：設定 API Key
  1. 在上方輸入框貼上您的 API Key
  2. 點擊 "儲存 API Key" 按鈕
  3. 等待狀態檢查完成 (顯示 ✅)
  4. API Key 會安全儲存在您的瀏覽器中
🤖 Step 3：使用 AI 分析
  1. 選擇分析模式 (新聞/社群/完整/深度)
  2. 點擊對應的按鈕
  3. 等待 AI 處理完成 (約 10-30 秒)
  4. 查看真實的 AI 分析報告
注意事項：
  • API Key 會安全儲存在您的瀏覽器本地
  • 不會上傳到任何伺服器
  • 每次使用都會消耗 Gemini API 配額
  • 建議定期更換 API Key 以確保安全
